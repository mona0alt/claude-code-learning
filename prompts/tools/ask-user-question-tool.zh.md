# AskUserQuestion 工具提示词

## 概述

向用户提出多项选择题以收集信息、澄清歧义、理解偏好、做出决定或提供选择。

## 何时使用

- 收集用户偏好或需求
- 澄清模糊的指令
- 在工作时获取关于实现选择的决定
- 为用户提供方向选择

## 使用说明

- 用户始终可以选择"Other"来提供自定义文本输入
- 使用 `multiSelect: true` 允许问题有多个答案
- 如果推荐特定选项，将其作为第一个选项并添加"(Recommended)"

## 预览功能

对于需要视觉比较的选项，使用可选的 `preview` 字段：
- UI 布局或组件的 ASCII 模拟
- 显示不同实现的代码片段
- 图表变体
- 配置示例

预览内容以等宽字体框中的 markdown 格式呈现。当任何选项有预览时，UI 切换到并排布局。

## 计划模式注意

在计划模式中，在最终确定计划**之前**使用此工具澄清需求或选择方案。

不要使用此工具问"我的计划准备好了吗？"或"我应该继续吗？" — 使用 ExitPlanMode 代替。

**重要：** 不要在问题中提及"计划"（例如"计划看起来好吗？"），因为用户在调用 ExitPlanMode 之前看不到计划。
