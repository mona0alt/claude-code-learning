# Write 工具提示词

## 概述

向本地文件系统写入文件。

## 关键说明

### 写入前

**如果是现有文件，必须先使用 Read 工具**读取文件内容。如果未先读取文件，此工具将失败。

### 重要规则

1. **此工具将覆盖**提供路径处的现有文件
2. **修改现有文件时优先使用 Edit 工具** — 它只发送 diff
3. **仅在以下情况使用 Write：**
   - 创建新文件
   - 完全重写
4. **除非用户明确要求，否则不要创建文档文件**（*.md）或 README 文件
5. **只有用户明确要求**才使用表情符号

### Write vs Edit 何时使用

| 情况 | 使用 |
|------|------|
| 修改现有文件 | **Edit** 工具 |
| 创建新文件 | **Write** 工具 |
| 完全重写现有文件 | **Write** 工具 |
