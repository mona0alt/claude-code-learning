# Read 工具提示词

## 概述

从本地文件系统读取文件。你可以直接使用此工具访问任何文件。

## 关键说明

### 基本用法

- `file_path` 参数必须是**绝对路径**，不是相对路径
- 默认从文件开头读取最多 2000 行
- 结果使用 `cat -n` 格式返回，行号从 1 开始

### 读取部分文件

- 可以选择指定行偏移和限制
- 当你知道需要文件的哪部分时，只读取那部分

### 特殊文件类型

**图片：**
- 可以读取 PNG、JPG 等图片格式
- 内容以视觉方式呈现（Claude Code 是多模态的）

**PDF：**
- 可以读取 PDF 文件
- 大型 PDF（10+ 页）：必须提供 pages 参数
- 示例：`pages: "1-5"`
- 每次请求最多 20 页

**Jupyter Notebooks：**
- 可以读取 .ipynb 文件
- 返回所有单元格及输出

### 重要说明

- 读取不存在的文件也可以（会返回错误）
- 如果文件存在但为空，会收到警告
- 只能读取文件，不能读取目录（用 Bash 的 `ls` 查看目录）
- 始终用此工具查看用户提供的截图

### 文件路径假设

- 假设此工具可以读取机器上的所有文件
- 如果用户提供路径，假设该路径有效
