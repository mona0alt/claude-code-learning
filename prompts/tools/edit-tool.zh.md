# Edit 工具提示词

## 概述

对文件执行精确的字符串替换。

## 关键说明

### 编辑前

**编辑前必须至少使用一次 Read 工具。** 如果未读取文件就尝试编辑，此工具将报错。

### 编辑格式

从 Read 工具输出编辑文本时：
- 保持**精确的缩进**（制表符/空格），与行号前缀后的格式相同
- 行号前缀格式是：`行号 + 制表符`
- 永远不要在 `old_string` 或 `new_string` 中包含行号前缀的任何部分

### 编辑规则

1. **优先编辑现有文件**，不要创建新文件
2. **除非明确要求，否则不要创建新文件**
3. **如果 `old_string` 在文件中不唯一，编辑将失败**
   - 提供更多上下文使其唯一，或
   - 使用 `replace_all` 更改所有实例
4. **使用 `replace_all`** 在文件中替换/重命名字符串

### 最小唯一性

使用最小的、能明确唯一的 `old_string` — 通常 2-4 行相邻文本就够了。避免包含 10+ 行上下文当较少内容就能唯一标识目标时。

### 表情符号策略

- 只有用户明确要求时才使用表情符号
- 除非被要求，否则避免向文件添加表情符号
