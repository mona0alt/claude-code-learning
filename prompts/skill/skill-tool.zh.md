# Skill 工具提示词

## 概述

在主对话中执行技能。

## 概述

技能提供专业能力和领域知识。当用户引用"斜杠命令"或"/<something>"时，他们指的是技能。

## 如何调用

使用此工具，传入技能名称和可选参数：

```typescript
// 示例：
skill: "pdf"                                    // 调用 pdf 技能
skill: "commit", args: "-m 'Fix bug'"          // 带参数调用
skill: "review-pr", args: "123"                // 带参数调用
skill: "ms-office-suite:pdf"                    // 完全限定名称
```

## 重要规则

1. **可用技能**列在系统提醒消息中
2. **当技能匹配**用户请求时，这是**阻塞性要求**：在生成任何其他响应**之前**调用技能
3. **永远不要**提及技能而不实际调用此工具
4. **不要调用**已经在运行的技能
5. **不要使用**此工具调用内置 CLI 命令（如 /help、/clear 等）
6. **如果你在当前轮次看到 `<command-name>` 标签**，技能已经被加载 — 直接按照说明操作

## 技能发现

相关技能每轮都会作为"与你任务相关的技能："提醒自动显示。如果你要做的事情不在其中：
- 调用 `DiscoverSkillsTool`，描述你要做什么
- 已显示或加载的技能会自动过滤
